## 部署方式
- 创建加密密钥`sh create_rsa_keys.sh`
- 启动gunicorn服务器`sh start_gunicorn.sh`，其参数在config.py中定义，其中在sh文件中需要写明gunicorn的路径
- 重建数据库`python run.py`，如遇权限问题可使用`sudo python run.py`，数据库文件路径为`app/database.db`
